01

Performance

The Fuel architecture is designed for high throughput parallel transaction execution using a novel UTXO approach. Boasting 7k TPS per core and 5k TPS on Ethereum (post-4844).

02

Sustainability

Fuel’s unique state model enables better state minimization through native rehydration, trading bandwidth for state access, which enables a far more sustainable and accessible set of node requirements for Fuel rollups.

03

Interoperability

Cross-rollup interoperability should be seamless for users. Fuel’s unique introspective block construction enables a new paradigm in cross chain communication.
